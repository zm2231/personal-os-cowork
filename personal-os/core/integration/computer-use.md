# Computer Use — Cowork Native Capability

Computer use is built into Cowork Pro/Max. No unlock required. Skills reference it where relevant.

## When Skills Use Computer Use

| Situation | What to Do |
|---|---|
| Calendar app on screen | Capture today's agenda before reading priorities file |
| Meeting app on screen | Capture agenda/attendees before processing notes |
| Error message screenshot shared | Extract text, identify type, suggest fix |
| "Organize my downloads" | Navigate Finder, categorize files by type |
| "Launch my work apps" | Open apps from working-preferences.md app list |
| Form on screen | Map fields to identity.md / context files, fill |
| "Analyze this screenshot" | Extract text + UI elements, describe content |

## Workflow Templates

Pre-built workflows in `personal-os/templates/workflows/`:
- `daily-setup.md` — workspace setup automation
- `downloads-organizer.md` — file organization workflow
- `form-filler.md` — form filling from memory

## Rules

- Always show what will change BEFORE changing it
- Never delete files without explicit confirmation
- For form submission: show filled values first, submit on approval
- Learn patterns: same screen 3+ times → suggest automation
