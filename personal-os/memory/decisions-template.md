# Decision Log

*Captures key decisions made during sessions. Auto-populated when you say "we decided X" or "I decided to..."*

## How to Use

Decisions are logged automatically — no commands needed. Say:
- "We decided to go with Option B"
- "I decided to push the deadline to April"
- "We're dropping the mobile feature for v1"

PersonalOS logs it briefly: "Logged."

## Decision Format

Each decision is logged as:

```
## D-[number] — [date]
**Decision:** [what was decided]
**Context:** [brief why, if stated]
**Assumptions:** [what must be true for this to hold]
**Review date:** [30–90 days out]
**Status:** Active / Superseded / Reviewed
```

## Reviewed / Superseded

*Decisions that have been revisited or replaced.*

**Last Updated:** [date]
