# Habit Tracker

Track your daily habits with consistency metrics and streak tracking.

## When to Use This Skill

Use when you want to:
- Track daily habits (exercise, reading, meditation, etc.)
- Monitor streaks and consistency
- Get motivated by progress visualization
- Identify which habits stick and which don't

## How It Works

### Level 1: Basic Tracking (Sessions 1-5)

I'll help you:
- Track up to 5 daily habits
- Record completion status each day
- Show simple streak counts
- Identify habits you're doing well with

**Example:**
```
User: Track my daily habits: meditation, exercise, reading, coding, journaling
User: I completed meditation, exercise, and coding today
User: How am I doing on my habits this week?
```

### Level 2: Advanced Metrics (Sessions 6-10)

I'll also help you:
- Track 6-10 habits
- Show weekly/monthly consistency rates
- Identify patterns (which days are hardest)
- Suggest habit optimization strategies
- Generate visual streak reports

**Example:**
```
User: Add "learning Spanish" to my habits
User: What's my weekly consistency for each habit?
User: Which day of the week am I most likely to skip habits?
User: How can I improve my consistency?
```

### Level 3: Habit Optimization (Sessions 11+)

I'll also help you:
- Unlimited habit tracking
- Advanced analytics and trends
- Habit stacking suggestions
- Automatic reminders and nudges
- Long-term goal alignment tracking

**Example:**
```
User: Analyze my habit trends over the last 3 months
User: Which habits should I stack together?
User: Remind me to do my morning routine
User: How do my habits align with my quarterly goals?
User: What habits should I drop or add?
```

## Data I Track

### Level 1
- Habits list (5 max)
- Daily completion status
- Current streak for each habit
- Best streak for each habit

### Level 2
- All Level 1 data
- Weekly consistency rate (percentage)
- Monthly consistency rate
- Day-of-week analysis
- Habit completion order

### Level 3
- All Level 2 data
- 3-month trend analysis
- Correlation between habits
- Success/failure factors
- Goal alignment metrics

## Best Practices

1. **Start small:** Track 3-5 habits at Level 1
2. **Be consistent:** Check in daily (I'll remind you)
3. **Focus on streaks:** Celebrate 7-day streaks
4. **Don't break the chain:** Missing one day resets streaks
5. **Adjust as needed:** Add/remove habits at the start of each week

## Tips for Success

1. **Start with 3 habits** before expanding to 5
2. **Check in daily** to maintain accurate streaks
3. **Celebrate 7-day streaks** - small wins build momentum
4. **Don't stress about missed days** - focus on consistency over perfection
5. **Adjust habits monthly** based on what's working

## Common Mistakes

- Tracking too many habits at once (start small)
- Not checking in daily (streaks reset)
- Being too hard on yourself (aim for 70%+ consistency)
- Keeping the same habits forever (evolve as you grow)
- Forgetting to celebrate progress (acknowledge wins!)
